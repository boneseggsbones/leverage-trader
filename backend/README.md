# Leverage Backend

This is the backend for the Leverage application.

## Getting Started

1.  Navigate to the `backend` directory:
    ```sh
    cd backend
    ```

2.  Install the dependencies:
    ```sh
    npm install
    ```

3.  Build the TypeScript code:
    ```sh
    npm run build
    ```

4.  Start the server:
    ```sh
    npm start
    ```

    The server will be running on `http://localhost:4000`.

## Development

For development, you can use the following command to automatically rebuild the code on changes and restart the server:

```sh
npm run dev
```
